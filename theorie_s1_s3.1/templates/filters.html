Démonstration des Filtres Jinja2

Les filtres transforment les données avant affichage avec le symbole |

Filtre UPPER

Code: {{ ville | upper }}

Résultat: {{ ville | upper }}

Description: Convertit le texte en MAJUSCULES

Filtre LOWER

Code: {{ ville | capitalize }}

Résultat: {{ ville | capitalize }}

Description: Met une majuscule au début et minuscules au reste

Filtre LENGTH

Code: {{ fruits | length }}

Résultat: {{ fruits | length }} fruits

Liste: {{ fruits }}

Description: Affiche le nombre d'éléments dans une collection

Filtre TRUNCATE

Code: {{ description | truncate(50) }}

Résultat: {{ description | truncate(50) }}

Original: {{ description }}

Description: Coupe le texte à N caractères et ajoute ...

Filtre DEFAULT

Code: {{ auteur | default("Auteur inconnu") }}

Résultat: {{ auteur | default("Auteur inconnu") }}

Description: Utilise une valeur de remplacement si la variable est None/vide

Filtre SORT

Code: {{ fruits | sort }}

Résultat: {{ fruits | sort }}

Description: Trie les éléments par ordre alphabétique

Filtre REVERSE

Code: {{ fruits | reverse | list }}

Résultat: {{ fruits | reverse | list }}

Description: Inverse l'ordre des éléments
